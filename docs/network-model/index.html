Burke's Theoremâ€‹

Wikipedia cite

Burke's theorem asserts that for a M/M/1 queue in the steady state with arrivals, a Poisson process with rate parameter Î»\lambdaÎ»:

  1. The departure process is a Poisson process with rate parameter Î»\lambdaÎ».
